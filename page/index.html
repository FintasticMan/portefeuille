InfiniTime

InfiniTime is an open-source smartwatch operating system written in C++, specifically for the PineTime. I've been working on improving the user experience by adding features I believe to be useful, and improving the development experience by improving things like the continuous integration systems since October 2021. In February 2023, I met up with the project lead, JF, at FOSDEM, a yearly meet-up in Brussels for many different free and open-source projects. A couple of weeks later, he asked if I would like to join the InfiniTime team!
